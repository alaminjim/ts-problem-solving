Qus no.1 --> টাইপ মূলত টাইপস্ক্রিপ্টের একটি নাম যা আমরা ডেটা টাইপ বর্ণনা করতে ব্যবহার করি। এর মাধ্যমে আমরা বুঝতে পারি যে ডেটার ধরন কী ধরণের হবে, এবং একটি অবজেক্ট বা অ্যারের ধরণও নির্ধারণ করতে পারি।

Interface হলো TypeScript এ মূলত object এবং class এর জন্য ব্যবহার করা হয়। Interface define করার সময় এটি সরাসরি value দিতে হয় না, কিন্তু object এর মতো ব্যবহার করা যায়। এর মাধ্যমে আমরা বলতে পারি কোন property তে কোন ধরনের মান থাকবে।

Qus no.3 --> any হলো এমন একটি type, যা তখন ব্যবহার হয় যখন আমরা কোন variable এর type declare করি না। এর মানে, এই variable এ যেকোনো ধরনের value assign করা যাবে, এবং TypeScript তার type check করবে না। তাই ভুল code ধরা পড়বে না।

--> unknown হলো এমন একটি type, যা জানায় যে value এর ধরন এখনো নির্ধারিত নয়। তাই আমরা যদি কোনো নির্দিষ্ট type ব্যবহার করতে চাই, তাহলে আগে type check বা validation করতে হবে।

--> never হলো এমন একটি type, যা কখনো কোনো value return করবে না। সাধারণত এমন function-এ ব্যবহার করা হয়, যা সবসময় error throw করে |
